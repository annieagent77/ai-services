## SEO Checklist

### Technical
- [ ] SSL certificate (HTTPS)
- [ ] Mobile responsive test
- [ ] Page speed >90
- [ ] XML sitemap
- [ ] robots.txt
- [ ] Schema markup (LocalBusiness)

### On-Page
- [ ] Title tags (unique per page)
- [ ] Meta descriptions
- [ ] H1/H2 hierarchy
- [ ] Image alt text
- [ ] Internal linking
- [ ] NAP consistency (Name, Address, Phone)

### Local SEO
- [ ] Google Business Profile linked
- [ ] Local keywords in content
- [ ] Service area pages
- [ ] Reviews integration
